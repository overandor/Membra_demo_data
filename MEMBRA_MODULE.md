# MEMBRA Module Contract — Demo Data

## Role

Seed-data package for MEMBRA demos. Provides coherent owners, advertisers, assets, media kits, relay jobs, proof events, scan events, wallet states, and KPI targets.

## System inputs

- demo scenario requirements
- pilot story requirements
- module fixture needs

## System outputs

- demo JSON bundles
- seed records
- sample KPI states
- proof/scanning/wallet examples

## Health

No runtime service required.

## Replit role

`data`

Used by `Membra_kpi`, the root MEMBRA website, and future Replit agent tasks to seed realistic product state.

## Production boundary

Demo records are not real customers, real payouts, or real campaign performance.
